## Theme  

The current theme uses [Hugo Theme Bootstrap](https://github.com/razonyang/hugo-theme-bootstrap/tree/master).  It has many features and is easy to override as-needed.

Checkout the [Hooks](https://hbs.razonyang.com/v1/en/docs/advanced/hooks/) feature to add customization without having to fully override theme layouts.
